SUBMODELS USED IN STUD.IO FILE
-----------------------------------------------------------------

1️⃣ **Base_Frame**  
   → 96 × 32 stud rectangular Technic beam grid  
   → Internal cross supports every 16 studs  
   → Reinforced with open Technic frames  

2️⃣ **Wall_Sections**  
   → Repetitive 1x2 bricks stacked to form 10-brick-tall walls  
   → Attached via Technic pin holes to frame  

3️⃣ **Bridge_Tower**  
   → SNOT structure + windows  
   → Sloped roof with wedge plates (user customizable)

4️⃣ **Hull_Number_68**  
   → Assembled using vertical SNOT bricks + white tiles  
   → Attached to side wall using jumper plates or brackets

5️⃣ **Radar_Top**  
   → Turntable base  
   → Bar elements for mast  
   → Round dish with radar decal (optional)
